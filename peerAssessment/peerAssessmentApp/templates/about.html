About New Instance


New Instance is a dynamic peer assessment software designed to aid in classrooms and work environments.The goal of our software is to help users effectively evaluate their peers or collegues, while keeping the reviews anonymous, and simultaneously allowing professors full access into these responses. The dynamic nature of the software should allow administrators to be able to use this for whatever type of project desired with full range of options to establish another project or course whenever necessary. New Instance is designed to be effective and easy to use for both users and administrators, and provides a helpful hand in the classroom and outside.
